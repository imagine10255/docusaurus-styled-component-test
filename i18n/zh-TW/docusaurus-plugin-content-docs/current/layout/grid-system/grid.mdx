# Grid

一個由容器控制項目如何分配的元件

相較於 Row Col，Grid 是由容器本身來設定內部項目的分配


:::caution Suggestion

Grid 元件並沒有要取代 Row Col，Row Col 負責 Layout排版，Grid 則負責卡片排版 與 列表排版

而他們水溝間距一樣都是 使用 `g-?`

:::

```tsx
import {Grid} from 'bear-react-grid';

<Grid>
    {/*...ignore some */}
</Grid>
```

```tsx title="types.ts"
export type RecordOption<K extends keyof any, T> = {
    [P in K]?: T;
};
export type TMediaSize = 'xs'|'sm'|'md'|'lg'|'xl'|'xxl'
```



### TGridGaps
```tsx
export type TGridGapUnit = 'px' | 'rem';
export type TGridGap = 0|`${number}${TGridGapUnit}`
```


```tsx title="breakpoints gap"
<Grid className="gap-2 gap-md-4"/>
<Grid className="column-gap-2 row-gap-md-4"/>
```




### TGridCols
```tsx
export type TGridCol = string|number|TGridColNumberSizeUnit|'min-content'|'max-content'|`minmax('${TGridColNumberSizeUnit}', '${TGridColNumberSizeUnit}')`;
export type TGridTemplate = TGridCol|RecordOption<TMediaSize, TGridCol>
```

```tsx title="auto calc child, example is repeat(3, auto)"
import {Grid, auto} from 'bear-react-grid';

<Grid col={auto(3)}>
    <div>TD 1</div>
    <div>TD 2</div>
    <div>TD 3</div>
</Grid>
```



```tsx title="auto x 6 td"
<Grid col={6}/>
```


```tsx title="1fr * 6"
import {fr} from 'bear-react-grid';

<Grid col={fr(6)}/>
```

```tsx title="Range"
import {minmax} from 'bear-react-grid';

<Grid col="minmax(10px, 1fr)"/>
<Grid col={minmax('10px', '1fr')}/>
```

```tsx title="1fr TD and 20px TD"
<Grid col="1fr 20px"/>
```

```tsx title="breakpoints"
<Grid col="1fr 1fr auto" md="1fr auto 1fr"/>
```
