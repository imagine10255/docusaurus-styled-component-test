# Flex

用來產生 FlexBox 的元件，但僅僅是為了讓之後維護時，方便閱讀 Html 結構


```tsx
import {Flex} from 'bear-react-grid';

<Flex className="flex-column align-items-center">
    {/*...ignore some */}
</Flex>
```

## Props

| Props Name  | Type                                                    | Required | Default | Description                                                   |
|-------------|---------------------------------------------------------|----------|---------|---------------------------------------------------------------|
| style       | CSS.Properties                                           |          |         |                                                               |
| className   | string                                                   |          |         |                                                               |
| col        | [TFlexDirection](#tflexdirection)  |          |     |   |
| sm         | [TFlexDirection](#tflexdirection)  |          |     |   |
| md         | [TFlexDirection](#tflexdirection)  |          |     |   |
| lg         | [TFlexDirection](#tflexdirection)  |          |     |   |
| xl         | [TFlexDirection](#tflexdirection)  |          |    |   |
| xxl        | [TFlexDirection](#tflexdirection)  |          |    |   |
| as         | elementType                         |          | `"div"` | You can use a custom element for this component                    |


### TFlexDirection
```ts
type TFlexDirection = 'column'|'row';
```


## Sample

```tsx
<Flex col="column" md="row" className="justify-content-between align-items-center gap-2">
    <div>Flex Child 1</div>
    <div>Flex Child 2</div>
    <div>Flex Child 3</div>
    <div>Flex Child 4</div>
</Flex>
```
